# GitHub Profile Cards

### Description

An app to render user info from GitHub API

### Technologies Used

- React.js
- JavaScript
- Axios

### Installation

```
git clone git@github.com:angelawong3/github-cards.git
cd github-cards
npm install

```

### Usage

```
npm start
```
